# Python Task Manager

This is a simple task manager I built using Python in IntelliJ IDEA.  
It’s a menu-based program that helps users stay organized by adding, viewing, completing, and deleting tasks.

## Overview

I created this project to practice real programming skills like problem-solving, structuring a program, and handling user input. The app runs in the terminal and keeps looping so the user can manage tasks continuously until they choose to exit.

## Features

- Add new tasks quickly
- View all current tasks in a list
- Mark tasks as completed
- Delete tasks
- Handles invalid input so the program doesn’t crash

## How It Works

The program uses a loop to keep running and display a menu.  
The user picks an option, and the program responds based on that choice.

Behind the scenes:
- A list stores all tasks
- A loop keeps the program running
- Conditionals decide what action to take
- Error handling prevents crashes if the user enters bad input

## How to Run

1. Open the project in IntelliJ IDEA
2. Open `main.py`
3. Click Run
4. Follow the menu options in the terminal
